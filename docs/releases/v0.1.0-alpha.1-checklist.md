# v0.1.0-alpha.1 release checklist

This is the auditable release record for Git tag `v0.1.0-alpha.1` and Python package version
`0.1.0a1`.

Publication evidence: [GitHub pre-release](https://github.com/xuu33030/agentbraid/releases/tag/v0.1.0-alpha.1)
and [tag CI run](https://github.com/xuu33030/agentbraid/actions/runs/29489071609).

## Source and governance

- [x] All milestone implementation issues are closed or represented by this release issue.
- [x] Every feature-branch commit contains a DCO `Signed-off-by` trailer.
- [x] The worktree is clean and the release candidate is pushed to `codex/v1-core`.
- [x] `LICENSE`, `NOTICE`, `THIRD_PARTY_NOTICES.md`, `SECURITY.md`, and contribution governance
  files are present.
- [x] README installation, architecture, host workflow, troubleshooting, and examples match the
  implemented alpha contract.
- [x] Draft PR is reviewed and merged to `main`.

## Provider and security boundary

- [x] `docs/provider-policy.md` was reviewed against current official Antigravity MCP, Skill, FAQ,
  and Codex non-interactive documentation on 2026-07-16.
- [x] AgentBraid has no code path that launches `agy` or reads Google/Antigravity credentials.
- [x] Provider subprocesses use argument arrays without shell execution and accept only the
  official `codex` executable name.
- [x] Runtime paths reject the repository and known credential-bearing directories.
- [x] Gitleaks 8.30.1 scanned the full release-candidate history with no findings.
- [x] `pip-audit . --strict --progress-spinner off` reported no known vulnerabilities.
- [x] The installed runtime dependency closure passed the allowlisted license audit.
- [x] GitHub private vulnerability reporting, secret scanning, push protection, and Dependabot
  security updates are enabled.

## Quality and distributions

- [x] Ruff lint and format checks pass.
- [x] Strict mypy checks pass for `src`, `tests`, and `scripts`.
- [x] 74 tests pass and branch-aware coverage exceeds the 80% gate at 85.83%.
- [x] GitHub Actions passes seven Linux, macOS, and Windows jobs for Python 3.11 through 3.13.
- [x] The wheel and source distribution build from an isolated environment.
- [x] Twine validates both distributions.
- [x] The wheel contains `py.typed`, Apache license files, and no tests, databases, or `.env`
  files.
- [x] A fresh environment installs the wheel without dependencies and reports
  `agentbraid 0.1.0a1`.
- [x] Final release archives and `SHA256SUMS` are generated from merged `main`.
